PLESK / DOMAIN KURULUM NOTU

Bu paket localhost adreslerini otomatik hale getirilmiş sürümdür.
Domain adını koda tek tek yazmana gerek yoktur.

1) Plesk'te Node.js app oluştur.
2) Proje klasörünü yükle.
3) Node.js ayarları:
   - Application root: proje klasörü
   - Application startup file: server.js
   - Node version: mümkünse 20+
4) Terminal / SSH varsa proje klasöründe çalıştır:
   npm install
   npm run build
5) Plesk Node.js ekranından uygulamayı restart et.
6) Domaini telefondan aç.

Sistem otomatik şunu yapar:
- Localde açılırsa API: http://localhost:4000
- Domain'de açılırsa API: mevcut domain adresi

Bu yüzden telefonda localhost sorunu yaşanmaz.

ÖNEMLİ:
3 adımlı yeni ekran bu pakete eklenmedi. Çalışan sistem bozulmasın diye sadece domain/API düzenlemesi yapıldı.
